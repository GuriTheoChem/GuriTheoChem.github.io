I built a lightweight, browser-based GUI for generating input files for the **HUMMR** quantum chemistry program.
Instead of writing inputs manually, you can now configure all keywords through a clean interface and instantly preview the generated file.
The tool includes organized sections for all major HUMMR keywords.

[:material-atom: HUMMR Input Builder](https://hummr-input-builder.streamlit.app){:target="_blank" .md-button .md-button--primary}

It’s fully [open source](https://github.com/GuriTheoChem/HUMMR-Input-Builder){:target="_blank"} and deployed on Streamlit Cloud.
